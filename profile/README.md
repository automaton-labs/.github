# Automaton Labs

Automaton Labs develops platforms and tools for working with agentic technology.

## Automatic

Automatic manages context, instructions and workflow for local agents, bringing improved productivity, ease of use, and cross-agent workflows.

- [https://tryautomatic.app](https://tryautomatic.app)
- [https://github.com/automaton-labs/automatic](https://github.com/automaton-labs/automatic)

## Skill.json

Skill.json is a simple standard for describing repositories of Skill files, for use with the Agentic Skills framework.

- [https://github.com/automaton-labs/skill.json](https://github.com/automaton-labs/skill.json)
